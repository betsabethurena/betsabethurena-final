# GreenTech Solutions Website

## Project Description
GreenTech Solutions is a multi-page website designed for a fictional startup focused on environmental sustainability and eco-friendly technology. The website showcases the company’s mission, products, educational resources, and contact information. It is designed to inform and engage users while promoting sustainable practices through technology.

The site includes interactive features such as product filtering, clickable content cards, and a contact form with validation.

---

## How to Run the Project

To run this project on your local machine:

1.  Clone the repository from GitHub: https://betsabethurena.github.io/betsabethurena-final/index.html

2. Open the project folder in Visual Studio Code.

3. Open the `index.html` file.

4. Run the project using one of the following methods:
- Open `index.html` directly in your web browser, OR
- Use the "Live Server" extension in VS Code for the best experience.
